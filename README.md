# Blog Engine

Blog engine demo.

Admin user password: **password**

# How to start the engine

```
yarn && yarn dev
```

Open browser http://localhost:3000

# Testing

Run E2E tests manually:

```
yarn && yarn e2e
```

Run unit tests manually:

```
yarn && yarn test
```

Run unit and E2E tests in CI mode:

```
yarn && yarn test:ci
```
